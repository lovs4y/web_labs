This repository contains lab work for the course Web Design. The labs demonstrate the development of a website for learning English using HTML, CSS, Bootstrap and JavaScript.

Project

The project developed during the lab sessions was a responsive form for creating integrations. It includes logic for login and registration, translation words when highlighting it, and for pages layout structured input fields for app details (such as name, bundle, platform, and tracker), checkboxes, and buttons.
